# self-workshop
这个项目目前还在开发中。

目前的想法是，将这个项目作为多个模块的集成平台，便利于使用者（主要是软件开发者）工作和学习使用。

目前确定要实现的模块有待办事项列表、个人博客。之后希望可以添加在线代码Playground、大模型对话。
